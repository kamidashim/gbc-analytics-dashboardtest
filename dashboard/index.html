⚙️ Подключение к Supabase

Дашборд работает с mock данными. Для подключения к реальному Supabase:
1. Задай переменные в .env.local: NEXT_PUBLIC_SUPABASE_URL и NEXT_PUBLIC_SUPABASE_ANON_KEY
2. Запусти node scripts/sync_to_supabase.js для первичной загрузки данных
3. Раскомментируй функцию loadFromSupabase() в dashboard/app.js
